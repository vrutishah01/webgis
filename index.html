ArcGIS Online - Web AppBuilder

ArcGIS Web AppBuilder create apps with ease by building web apps without writing a single line of code. It includes tools to configure fully featured HTML apps.

The Land use land cover (LULC) map for ‘Isle of Wight’ (IOW) region has generated using ArcGIS Web AppBuilder in ArcGIS online platform. The data layers used in the map are location of towns, minor roads, major roads, average rainfall grid and soilscapes showing the characteristics of soil. By clicking on the each feature layer, the information regarding the feature will pop-up. For instance, the name of the town, latitude, longitude, and population information will pop-up by clicking on ‘towns’ layer. The widgets are available to zoom-in and zoom-out the map and to search the address or place. The coordinate information, scale bar, legend, and attribute table are also available in the app

JavaScript API – ESRI Web Map

The web map has created using ArcGIS API for JavaScript approach. The ArcGIS API for JavaScript is a way to embed maps and tasks in web applications. JavaScript API is hosted by ESRI and available to developers in several configurations. The web map represents the region of ‘Isle of Wight’ (IOW) having heavy rainfall comparatively, using rainfall grid layer which represents the information regarding average rainfall of the specific area by clicking on the grid. The web map also shows the town locations at the region of heavy rainfall. The third data layer shows the characteristics or types of soil drainage within the island extent. Total fiver types of soil drainage categories can be seen, that are; freely draining, impeded drainage, naturally wet, slightly impeded drainage and others. By clicking on the each data layer the pop-up shows the information regarding the feature.

Geoserver - Leaflet API map

GeoServer is an open source server for sharing geospatial data. It is designed for interoperability and excels at publishing any major spatial data source using open standards. Leaflet is the leading open-source JavaScript library to construct interactive maps that works efficiently across all major desktop and mobile platforms, can be extended with lots of plugins, has a well-documented API. This web map has created by setting up GeoServer with leaflet JavaScript API. The web map represents the location of the Newport, Ryde and Sandown. These towns are high prone to flood comparatively than others and nearer to coast line of the island. The coastal boundary of the island has also been represented in the map.

Google KMZ

The web map has created by displaying the information of KML file in a Google map. For this procedure, Google maps JavaScript API approach has applied. It is worth to note that, the KML file has stored in GitHub online platform to load the KML layer in the code. The output web map shows the major roads (KML layer) of the ‘Isle of Wight’.
